# 智能停车管理系统

一个基于人工智能和物联网技术的现代化停车解决方案，提供用户端和管理端两大功能模块。

## 项目特色

### 🎨 现代化设计
- **响应式设计**：完美适配桌面、平板和手机设备
- **视觉美学**：采用蓝色系渐变配色，现代简约风格
- **动画效果**：丰富的页面动画和交互效果
- **用户友好**：直观的界面设计和操作流程

### 🔧 核心功能

#### 用户端功能
- 🏠 **首页仪表板**：系统概览和快速入口
- 🔍 **停车场查询**：智能搜索和地图定位
- 📅 **预约管理**：在线预约车位和时间管理
- 💳 **缴费中心**：多种支付方式和发票管理
- 👤 **个人中心**：用户信息和车辆管理

#### 管理员端功能
- 📊 **数据仪表板**：实时数据统计和图表分析
- 👥 **用户管理**：用户信息管理和权限控制
- 🏢 **停车场管理**：停车场信息编辑和状态监控
- 📅 **预约管理**：预约订单处理和状态更新
- 💳 **支付管理**：交易记录和财务统计
- 🔧 **设备管理**：硬件设备状态监控和维护
- 🚨 **异常处理**：系统异常报警和处理
- ⚙️ **系统设置**：系统配置和参数管理

#### 设计亮点

## 快速开始

### 本地运行
1. 下载项目文件到本地目录
2. 使用Python启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问：`http://localhost:8000`

## 部署说明

### 静态部署
项目为纯前端应用，可部署到任何支持静态文件托管的平台：

- **Nginx**：高性能Web服务器
- **Apache**：传统Web服务器
- **CDN**：内容分发网络
- **云存储**：阿里云OSS、腾讯云COS等

### 生产环境
建议使用专业的Web服务器部署，配置HTTPS和GZIP压缩：

```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        root /path/to/your/project;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
    
    # 启用GZIP压缩
    gzip on;
    gzip_types text/html text/css application/javascript;
}
```
